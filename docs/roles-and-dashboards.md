# Roles and Dashboards

## Student Dashboard
- Access lessons, quizzes, and virtual labs.
- Track progress and view teacher feedback.
- Works offline; syncs when connected.

## Parent / Guardian Dashboard
- Monitor child's attendance and academic progress.
- Receive alerts for performance risks.
- See teacher comments and recommendations.

## Teacher Dashboard
- Manage class roster and assignments.
- Create and grade assessments.
- Monitor student progress and provide interventions.

## Principal Dashboard
- Monitor school-wide student performance.
- Track teacher activity and class outcomes.
- Identify schools in need of additional support.

## County Education Office Dashboard
- Compare performance across schools.
- Access infrastructure and engagement metrics.
- Identify trends and gaps.

## Ministry Dashboard
- National performance overview.
- Curriculum adoption and impact metrics.
- Audit trails and compliance logs.
