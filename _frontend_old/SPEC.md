# LiberiaLearn MVP Specification

## Vision
LiberiaLearn is a national Grade 1–12 education system designed to raise Liberia’s academic standards to the level of Japan, Korea, and China. 
It focuses on STEM excellence while keeping full subject coverage: literacy, civics, arts, and career readiness.

## Guiding Principles
- Teacher-first design.
- Mastery-based progression.
- Localized bilingual content (en-LR + Koloqua).
- Low-bandwidth & offline-first.
- Data-driven continuous improvement.

## MVP Goals
1. Core classroom operations (attendance, assignments, gradebook, messaging).
2. Curriculum structure: Standards → Units → Lessons → Practice → Assessments.
3. Seed sample content for:
   - G1 Math (Counting → Addition ≤ 10)
   - G1 Reading (Phonics → Sight Words)
   - G7 Algebra (Linear Equations)
   - G10 Physics (Motion → Force → Energy)
4. Analytics dashboards for mastery tracking.
5. Guardian mobile view (attendance + homework + messages).

## Definition of Done
- Teachers can plan lessons, mark attendance, assign & grade work.
- Students can view lessons, complete practice, track mastery.
- Guardians can see progress on any phone.
- All pages usable on 360 px width screens.
